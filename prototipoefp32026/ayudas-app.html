GRUPO 6

MDI

En el MDI encontramos las opciones principales con las que cuenta el sistema. Cada opcion permitira realizar una serie de actividades las cuales tienen como objetivo hacer que el sistema sea facil de utilizar.

AYUDA

La opcion de ayuda tiene la finalidad de dar a conocer el funcionamiento correcto del sistema. Esto con el fin de proporcionar una herramienta que solucione las dudas del usuario relacionadas al sistema.

PROCESOS - MANTENIMIENTOS

En la opcion procesos encontraremos formularios, uno de ellos Asignacion aplicaciones a Perfil donde podremos obtener informacion al agregar el codigo de un perfil, ademas, podremos seleccionar la aplicacion para ver los permisos.
