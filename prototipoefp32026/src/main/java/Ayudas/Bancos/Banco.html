Clase Banco

La clase Banco representa una institución bancaria dentro del sistema.

Funcionamiento:

  1. Se crea un banco con un nombre obligatorio.
  2. Puede tener dirección, teléfono y correo.
  3. Se registra automáticamente la fecha de creación.
  4. Sirve como referencia para las cuentas bancarias.

Campos:

  • Banid: Identificador único
  • Bannombre: Nombre del banco
  • Bandireccion: Dirección
  • Bantelefono: Teléfono
  • Bancorreo: Correo
  • Banfecharegistro: Fecha de registro
