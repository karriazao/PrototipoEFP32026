Clase Movimiento Bancario

Registra todas las transacciones realizadas en una cuenta.

Funcionamiento:

  1. Se genera un movimiento por cada transacción.
  2. Puede ser depósito, retiro, pago, etc.
  3. Se registra el monto y descripción.
  4. Se vincula a una cuenta bancaria.
  5. Puede marcarse como conciliado o no.

Campos:

  • Movbid
  • Movbfechamovimiento
  • Movibmonto
  • Movdescripcion
  • CBANid
  • TTid
  • Movbtipomov
  • Movbreferencia
  • Movbconciliado
