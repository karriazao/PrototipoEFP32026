Clase Tipo de Cuenta

Define los tipos de cuentas que existen en el sistema.

Funcionamiento:

  1. Se crean diferentes tipos de cuenta (ej: Ahorro, Monetaria).
  2. Cada tipo tiene un nombre único.
  3. Se puede agregar una descripción.
  4. Las cuentas bancarias usan este catálogo para clasificarse.

Campos:

  • TCidcuenta: Identificador único
  • TCnombretipo: Nombre del tipo
  • TCdescripcion: Descripción
