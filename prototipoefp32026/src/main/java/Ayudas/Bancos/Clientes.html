Clase Cliente

Representa a una persona o entidad que posee cuentas bancarias.

Funcionamiento:

  1. Se registra un cliente con nombre obligatorio.
  2. Puede tener NIT único.
  3. Se guarda información de contacto.
  4. El cliente puede tener múltiples cuentas.

Campos:

  • Clid
  • Clinombre
  • Clinit
  • Clitelefono
  • Cliestado
  • Clidireccion
  • Clicorreo
