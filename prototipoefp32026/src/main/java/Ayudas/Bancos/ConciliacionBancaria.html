Clase Conciliación Bancaria

Permite comparar el saldo del sistema con el saldo del banco.

Funcionamiento:

  1. Se toma el saldo del sistema.
  2. Se compara con el saldo del banco.
  3. Se calcula la diferencia.
  4. Se asigna un estado (conciliado o no).

Campos:

  • Conbid
  • Conbfecha
  • Conbsaldosistema
  • Conbsaldobanco
  • Conbdiferencia
  • CBANid
  • Catesid
