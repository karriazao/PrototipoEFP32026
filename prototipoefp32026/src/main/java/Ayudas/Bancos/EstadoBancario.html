Clase Estado de Conciliación

Define los estados posibles de una conciliación bancaria.

Funcionamiento:

  1. Se crean estados como Conciliado, Pendiente o Con Diferencia.
  2. Cada estado es único.
  3. Se utiliza en la conciliación bancaria.
  4. Permite saber si los saldos coinciden o no.

Campos:

  • Catesid: Identificador
  • Catesnombreestado: Nombre del estado
