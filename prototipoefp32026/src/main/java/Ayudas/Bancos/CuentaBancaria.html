Clase Cuenta Bancaria

Representa una cuenta asociada a un cliente y un banco.

Funcionamiento:

  1. Se crea con un número de cuenta único.
  2. Se asigna a un cliente y banco.
  3. Tiene un saldo inicial.
  4. Se registra la fecha de apertura.
  5. Se relaciona con un tipo de cuenta.

Campos:

  • CBANid
  • CBANnumerocuenta
  • CBANsaldoactual
  • CBANfechaapertura
  • Banid
  • Cliid
  • TCidcuenta
