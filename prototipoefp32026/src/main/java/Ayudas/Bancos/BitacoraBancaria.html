Clase Bitácora Bancaria

Registra todas las acciones realizadas en el sistema.

Funcionamiento:

  1. Guarda quién hizo la acción.
  2. Indica qué acción se realizó.
  3. Registra la tabla afectada.
  4. Guarda valores antes y después.
  5. Se almacena la fecha automáticamente.

Campos:

  • BBid
  • BBusuarioaccion
  • BBaccion
  • BBtabla
  • BBregistroid
  • BBvaloranterior
  • BBvalornuevo
  • BBfechaaccion
  • BBdescripcion
